# 课程框架

以 Java 技术为基础开发出一个简单的 Web 项目

## 相关技术

- Java: 编程语言
- Spring Boot: 应用框架
- MySQL: 关系型数据库
- Tomcat: 服务器
- Linux: 操作系统
- Maven: Java项目管理工具
- Git: 代码管理工具
- HTML/CSS/JavaScript: 前端网页开发语言
- Chrome: 浏览器

## 基本概念

- 浏览器 <-> 网络 <-> 服务器 <-> 应用程序 <-> 数据库
- CRUD
- MVC：数据/呈现/逻辑
